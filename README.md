# Custom-Bootloader-Bare-metal-Drivers-on-STM32F103C8T6
Lightweight Custom Bootloader for STM32F103 enabling UART Firmware Updates (XMODEM) via Vector Table relocation. Features efficient Bare-metal Drivers (GPIO, USART, TIM) written in pure C, completely bypassing HAL. Demonstrates deep control over ARM Cortex-M3 memory mapping and register-level programming.
